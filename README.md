### Features / Highlights

Key Features:
KPI Tracking: Sales, profit, discounts, and order metrics.

Geographic Analysis: Sales distribution by country.

Trend Visualization: Monthly and daily sales performance.

Payment Insights: Breakdown of sales by payment method.

Category Performance: Top-performing product segments.

Technical Implementation:

Data Processing: Cleaned and transformed raw data in PostgreSQL for accuracy.

Dashboard Development: Built in Power BI with interactive filters for dynamic analysis.

Automation: Streamlined data pipelines for efficient updates.

Business Impact:
Identifies high-performing regions and underperforming markets.
Helps optimize discounts to maximize profitability.
Enhances strategic decisions on inventory and payment preferences.
